# PULZEN - O spaceshooter inovador!

**PULZEN** é um jogo estilo spaceshooter, onde o fundo do jogo é o seu computador! Além de ter poderes para serem utilizados durante sua jogatina!

## JOGABILIDADE
- Use o *mouse* para mirar de forma vertical.
- Atire usando o *botão esquerdo do mouse*.
- Use os *números do teclado* para usar os poderes.

## PODERES DISPONÍVEIS
- **Feijão:** paraliza os inimigos por *dois* segundos

## COISAS QUE FALTAM PARA O JOGO
- Mais poderes, como rifle (aumentar o fire rate) e outros.
- Variações de monstros.
- Sprites mais detalhados.
